OK
IchigoJam BASIC 1.2b53～
IchigoJam BASIC RPi
IchigoJam PC 非対応
-----------------------------------------------------------
[ 機能 ]
プログラム終了時の「OK」やエラーの表示・非表示を切り替えます。

[ コマンド 書式 ]
OK [<表示>]

[ 例 ]
OK 2

[ 解説 ]
<表示> は次のとおりです。デフォルトは 2 です。

    0・省略 変更なし（無視）
    1       非表示
    2       表示

0.9.4（正式版 0.9.7）～1.2b52（正式版 1.2.3）では
意図せず OK のみで「OK」の表示なく終了する動作になっていました。
（OK で Enter を押してもエラーにならないよう対応した時に同時発生）
これが正式にコマンド対応されています。
