HELP
1.0.1〜
-----------------------------------------------------------
[ 機能 ]
メモリマップを表示します。

[ コマンド 書式 ]
HELP

[ 例 ]
HELP

[ 解説 ]
下記の内容が表示されます。

MEMORY MAP
#000-#6FF CHAR PATTERN(ROM)
#700-#7FF CHAR PATTERN(RAM)
#800-#8FF VARS
#900-#BFF VRAM
#C00-#FFF PROGRAM

1.1 beta6 では容量対策として一時的に削除されています。
1.1 beta7 で復帰しています。
