INPUT
0.6〜
-----------------------------------------------------------
[ 機能 ]
入力を促し、変数へ入れます。

[ コマンド 書式 ]
PRINT ["<文字列>",]<変数>

[ 例 ]
INPUT "ANS?",A

[ 解説 ]
Enter(return) が入力されるまで数値の入力を促します。
入力された数値は <変数> へ入ります。
<文字列> がある場合は表示を行い、続けて入力となります。
ない場合は ? を表示します。

通常はキーボードからの入力を行いますが、
シリアル接続を行っている場合は、シリアルからの入力を受け付けます。
